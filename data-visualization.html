1.2 21369565

  • add 'max-xfer-size' property into sd driver for HGST SSD specifically
  • target to 200G, 400G and 1.6T capacity disks
  • set 128k as default value of max transfer size
  • ZFS makes use of this property to improve I/O performance in afp product

1.3 Benefit

  • avoid starving of small I/O (64k ~ 128k)
  • improve I/O stability of throughput

4 Automation

4.1 Combine all parts via script language

4.1.1 What need to be automated

  • filter raw data to generate data we care
  • configure visualization tool
  • generate visualization result (e.g., png file)
  • archive testing result

4.1.2 Which language

  • Perl, Ruby and Python etc.
  • sed, awk

  2. plot with Gnuplot. Ruby invoke Gnuplot example: 
    1: IO.popen("gnuplot", "w") { |pipe|
2:   pipe.puts("set term png")
3:   pipe.puts("set output 'sin.png'")
4:   pipe.puts("plot sin(x)")
5:   pipe.puts("quit")
6: }
7: 
8: system("display sin.png")
